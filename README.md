# 🎓 Student Segmentation & Personalized Course Recommendation System

## 📌 Project Overview
This project focuses on building a **data-driven personalization system** for an online learning platform (EduPro).  
It segments learners based on their behavior and recommends relevant courses.

---

## 🚀 Features
- Learner segmentation using K-Means clustering
- Feature engineering (engagement, preference, behavioral)
- Personalized course recommendation system
- Interactive Streamlit web application

---

## 📊 Datasets Used
- Users Dataset
- Courses Dataset
- Transactions Dataset
- Teachers Dataset

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Matplotlib, Seaborn

---

## 🧠 Methodology
1. Data Preprocessing
2. Feature Engineering
3. Clustering (K-Means)
4. Recommendation System
5. Streamlit App Development

---

## 📈 Results
- Successfully segmented learners into meaningful groups
- Improved recommendation relevance
- Demonstrated personalized learning paths
